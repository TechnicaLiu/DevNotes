# 关于React

## 简介

>用于构建用户界面的JavaScript库   （将数据渲染到为HTML视图）

## 特点

- 采用**组件化模式**，声明式编码，提高开发效率及组件复用率。

- 在`React Native` 中可以使用 React  语法进行**移动端开发**。

- 使用虚拟DOM + Diffing算法，尽量减少与真实Dom的交互。

## 相关js库

- react.js react 核心库

- react-dom.js ：提供操作DOM 的react扩展库

- babel.min.js ：解析成JSX语法代码转为 JS 代码的库
